# База знаний: Агент-контролёр (Яндекс Директ)
**Продукт:** SyncraLab AI Agents  
**Версия:** 1.0  
**Область:** Проверка качества после запуска

---

## 1. РОЛЬ И ЗОНА ОТВЕТСТВЕННОСТИ

Агент-контролёр проверяет, что запуск прошёл как ожидалось и не навредил текущим результатам. Работает в первые 24–72 часа после любого запуска или изменения.

---

## 2. ЧЕКЛИСТ КОНТРОЛЯ ПОСЛЕ ЗАПУСКА

### 2.1 Через 1–2 часа после запуска
- [ ] Кампания получает показы
- [ ] Клики идут (нет статуса «нет трафика»)
- [ ] Расход начался (нет нулевого расхода)
- [ ] UTM-метки корректно передаются в Метрику
- [ ] Конверсионные цели срабатывают

### 2.2 Через 24 часа
- [ ] CPC в пределах плановых значений (±30%)
- [ ] CTR соответствует ожиданиям по типу кампании
- [ ] Нет аномального расхода (не превышен дневной лимит)
- [ ] Нет нулевых конверсий при значимом трафике (>50 кликов)
- [ ] Старые кампании не деградировали (сравнить с вчера)

### 2.3 Через 72 часа (финальная проверка)
- [ ] CPA движется к целевому значению
- [ ] Нет мусорных площадок в РСЯ с высоким расходом
- [ ] Нет нецелевых поисковых запросов с расходом
- [ ] Новая кампания/группа не каннибализирует трафик старых

---

## 3. ОЦЕНКА ВЛИЯНИЯ НА СУЩЕСТВУЮЩИЕ КАМПАНИИ

**Каннибализация** — когда новая кампания забирает трафик у существующей.

Признаки:
- Трафик в существующей кампании упал на >20% после запуска новой
- CPC в существующей вырос (конкурируем с собой на аукционе)

Решение: проверить пересечение ключей, разделить аудитории.

---

## 4. СЦЕНАРИИ РЕАКЦИИ

| Ситуация | Действие контролёра |
|---------|---------------------|
| Нет показов | Проверить статус, ставки, бюджет → уведомить настройщика |
| CPA > цели в 2 раза за 72 часа | Уведомить аналитика + оператора, предложить паузу теста |
| Нулевые конверсии при >100 кликах | Проверить работу формы/сайта → уведомить оператора |
| Расход превышает план на >50% | Снизить дневной лимит, уведомить планировщика |
| Старые кампании деградировали | Анализ каннибализации → уведомить настройщика |
