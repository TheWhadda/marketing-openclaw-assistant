# База знаний: Агент-копирайтер (Яндекс Директ)
**Продукт:** SyncraLab AI Agents  
**Версия:** 1.0  
**Область:** Написание текстов рекламных объявлений

---

## 1. РОЛЬ И ЗОНА ОТВЕТСТВЕННОСТИ

Агент-копирайтер создаёт тексты объявлений, которые точно соответствуют запросу пользователя, передают УТП клиента и мотивируют к клику.

**Делает:**
- Пишет заголовки и тексты объявлений для поиска
- Пишет тексты для РСЯ и Мастер кампаний
- Создаёт быстрые ссылки, уточнения, описания
- Готовит варианты для A/B тестирования
- Адаптирует тексты под аудиторные инсайты от дипресёрчера

**Не делает:** не собирает семантику, не создаёт баннеры, не настраивает кампанию.

## 2. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ ЯНДЕКС ДИРЕКТ

### 2.1 Текстово-графическое объявление (поиск)

| Элемент | Лимит | Обязательно |
|---------|-------|------------|
| Заголовок 1 | 56 символов (с пробелами) | Да |
| Заголовок 2 | 30 символов | Нет, но рекомендуется |
| Текст объявления | 81 символ | Да |
| Отображаемый URL (путь) | 20 символов каждый из 2 полей | Нет |

### 2.2 Расширения

| Расширение | Лимит | Рекомендация |
|-----------|-------|-------------|
| Быстрые ссылки | 4 штуки, заголовок ≤ 30 символов, описание ≤ 60 символов | Использовать всегда |
| Уточнения | До 8, каждое ≤ 25 символов | Использовать всегда |
| Цена | До 8 позиций | Если есть конкретные тарифы |
| Контактная информация | Адрес, телефон | Для локального бизнеса |

### 2.3 РСЯ / Мастер кампаний

| Элемент | Лимит |
|---------|-------|
| Заголовок | 56 символов |
| Текст | 81 символ |
| Изображение | готовит иллюстратор |

## 3. ФОРМУЛЫ СИЛЬНОГО ЗАГОЛОВКА

### 3.1 Основные формулы

**Формула 1: Запрос + УТП**
```
[Что ищет пользователь] + [Ваше главное преимущество]
«CRM для бизнеса — бесплатный старт»
```

**Формула 2: Результат + Срок**
```
[Желаемый результат] + [За какое время]
«Сайт за 3 дня — гарантия результата»
```

**Формула 3: Проблема → Решение**
```
[Боль аудитории] → [Как вы её закрываете]
«Теряете клиентов? Автоматизируйте продажи»
```

**Формула 4: Конкретная цифра**
```
[Цифра] + [Что получает клиент]
«5 000 клиентов уже работают с нами»
```

**Формула 5: Вопрос-триггер**
```
[Вопрос о потребности]
«Нужна CRM под ваш бизнес?»
```

### 3.2 Что делает заголовок кликабельным

✅ Точное вхождение ключевого слова (подсвечивается жирным в выдаче)  
✅ Конкретная цифра (цена, срок, количество)  
✅ Сильный глагол действия  
✅ УТП, которого нет у конкурентов  
✅ Ответ на вопрос пользователя  

❌ Расплывчатые обещания («качественный сервис», «профессиональный подход»)  
❌ Превосходные степени без доказательства («лучший», «самый»)  
❌ Штампы («широкий ассортимент», «гибкие условия», «индивидуальный подход»)

## 4. СТРУКТУРА ТЕКСТА ОБЪЯВЛЕНИЯ

### 4.1 Формула текста (81 символ)

```
[Ключевая выгода] + [Снятие возражения или доп. факт] + [CTA]
```

**Пример:**
```
Интеграция за 1 день, обучение бесплатно. Попробуйте 14 дней бесплатно.
```

### 4.2 Типы CTA (призыв к действию)

| Тип CTA | Когда использовать |
|---------|-------------------|
| «Оставьте заявку» | Услуги, длинный цикл сделки |
| «Узнайте цену» | Когда цена — фактор выбора |
| «Попробуйте бесплатно» | SaaS, пробный период |
| «Рассчитайте стоимость» | Сложные индивидуальные услуги |
| «Закажите сейчас» | E-commerce |
| «Позвоните» | Когда звонок = конверсия |

## 5. БЫСТРЫЕ ССЫЛКИ

Каждая быстрая ссылка — мини-объявление. Принципы:

1. **Не дублировать** основной текст — расширять предложение
2. **Конкретность:** «Тариф Старт — 990₽/мес» лучше чем «Тарифы»
3. **Разные потребности:** покрывать разные сегменты аудитории
4. **Активный глагол или существительное с цифрой**

**Пример для CRM:**
- «Тариф для команды до 10 чел.»
- «Интеграция с 1С и Телефонией»
- «Демо за 30 минут — бесплатно»
- «Отзывы 500+ клиентов»

## 6. АДАПТАЦИЯ ПОД АУДИТОРНЫЕ СЕГМЕНТЫ

Разные группы объявлений — разные тексты под разную аудиторию.

| Сегмент | Акцент в тексте |
|---------|----------------|
| Малый бизнес | Простота, быстрый старт, цена |
| Средний бизнес | Масштабируемость, интеграции, поддержка |
| Крупный бизнес | Безопасность, SLA, кастомизация |
| B2C (срочная потребность) | Скорость, гарантия, простота |
| B2C (плановая покупка) | Сравнение, выбор, доверие |

## 7. A/B ТЕСТИРОВАНИЕ ТЕКСТОВ

### 7.1 Что тестировать (по приоритету)
1. Заголовок 1 (максимальное влияние на CTR)
2. Оффер (влияет на CR после клика)
3. CTA (влияет на намерение)
4. Расширения (влияют на CTR и качество трафика)

### 7.2 Правила теста
- Тестировать **один элемент** за раз
- Минимум **100 кликов на каждый вариант** для вывода
- Победитель определяется по CTR × CR (не только CTR)
- Всегда иметь контрольный вариант

### 7.3 Гипотезы для текстов

| Гипотеза | Тест |
|---------|------|
| Цена в заголовке повышает CTR | Заголовок с ценой vs без цены |
| Гарантия снижает возражение | Текст с гарантией vs без |
| Вопрос в заголовке привлекает внимание | Вопрос vs утверждение |
| Конкретная цифра лучше абстракции | «500 клиентов» vs «крупная база» |

## 8. ЗАПРЕЩЁННЫЕ ПРИЁМЫ (МОДЕРАЦИЯ)

| Нарушение | Пример | Решение |
|----------|--------|---------|
| Контактные данные в тексте | «Звоните: 8-800-...» | Убрать, использовать визитку |
| Превосходные степени | «Лучший на рынке» | Добавить подтверждение или убрать |
| Гарантии без оснований | «100% результат» | Конкретизировать или убрать |
| Призыв к срочности с обманом | «Только сегодня!» (не правда) | Только реальные акции |
| Прямое сравнение с конкурентом | «Лучше, чем Битрикс» | Убрать |

## 9. ИСПОЛЬЗОВАНИЕ АУДИТОРНЫХ ИНСАЙТОВ

Копирайтер получает от дипресёрчера:
- Дословные формулировки болей → использовать в тексте буквально
- Возражения → снимать в тексте («Боитесь сложной интеграции? За 1 день всё настроим»)
- Критерии выбора → превращать в УТП

**Принцип:** лучший текст объявления — это слова самого клиента.
